Déploiement sur GoDaddy

Option 1 — GoDaddy Hosting (cPanel)

  1. Connectez-vous à votre compte GoDaddy → Mes produits
  2. Cliquez sur Hébergement Web → Gérer
  3. Ouvrez le Gestionnaire de fichiers dans cPanel
  4. Allez dans le dossier public_html
  5. Uploadez TOUS les fichiers du dossier medipharma.ca/
  6. Votre site est en ligne sur votre domaine !

Option 2 — Netlify (plus rapide, gratuit)

  1. Créez un compte gratuit sur netlify.com
  2. Glissez-déposez le dossier medipharma.ca/ sur Netlify
  3. Le site est en ligne en 30 secondes
  4. Dans GoDaddy : allez dans DNS → pointez vers Netlify
  5. Dans Netlify : ajoutez votre domaine GoDaddy dans Domain settings
